# spring-tool-suite.nix

## Nix expression for spring tool suite

This is just a simple expression that uses prebuilt binary from https://spring.io/tools

Note that the built binary have not been fully tested.

## Installation
```
nix-env -f sts.nix -i
```
or
```
nix-build sts.nix
```

## Usage
```
STS
```
or
```
./result/bin/STS
```
